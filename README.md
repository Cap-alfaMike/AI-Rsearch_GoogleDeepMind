# Google DeepMind: AI Research Foundations — Study Repository

This repository documents my learning journey through the **Google DeepMind: AI Research Foundations** learning path on **Google Skills Boost**, managed by **Google Cloud**.

The curriculum focuses on the foundations behind modern language models, including small language models, text representation, neural network training, transformer architectures, fine-tuning, and model acceleration.

> This is a personal study repository created to organize my notes, labs, experiments, and implementation exercises while following the course.

The goal of this repository is to consolidate my learning process and demonstrate practical understanding of how modern language models are built, trained, adapted, and optimized.

## Learning Path

The course path includes the following activities:

### 01. Build Your Own Small Language Model

Introduction to the machine learning development pipeline for language models.

Topics studied:

- Language modeling fundamentals;
- N-gram models;
- Small Language Models;
- Strengths and limitations of traditional language models;
- Comparison between n-gram models and transformer-based models;
- High-level language model development pipeline.

---

### Skill Badge: Train A Small Language Model

Practical skill badge focused on training a small language model.

Topics studied:

- Formulating language model research problems;
- Building a simple tokenizer;
- Preparing a dataset for training;
- Training a transformer language model;
- Evaluating basic model behavior.

---

### 02. Represent Your Language Data

Focus on how raw text becomes usable input for machine learning models.

Topics studied:

- Text preprocessing;
- Tokenization;
- Vocabulary construction;
- Numerical representation of text;
- Dataset preparation;
- Input formatting for language models.

---

### 03. Design and Train Neural Networks

Focus on the training process for neural networks.

Topics studied:

- Neural network design;
- Training loops;
- Loss functions;
- Optimization;
- Overfitting;
- Underfitting;
- Generalization;
- Training diagnostics.

---

### 04. Discover The Transformer Architecture

Introduction to the transformer architecture behind modern language models.

Topics studied:

- Transformer architecture;
- Self-attention;
- Attention weights;
- Context-sensitive next-token prediction;
- Prompt processing;
- Attention visualization;
- Positional information.

---

### 05. Fine-Tune Your Model

Focus on adapting pre-trained models to specific tasks.

Topics studied:

- Full-parameter fine-tuning;
- Task-specific adaptation;
- Dataset formatting for fine-tuning;
- Model evaluation after fine-tuning;
- Practical limitations of fine-tuning;
- Fine-tuning small language models.

---

### 07. Accelerate Your Model

Focus on making model training and inference more efficient.

Topics studied:

- GPU acceleration;
- Hardware-aware training;
- Memory constraints;
- Efficient training;
- Training larger models with limited compute;
- Performance optimization.
